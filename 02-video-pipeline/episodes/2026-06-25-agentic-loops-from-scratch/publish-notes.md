# Publish notes: Agentic loops from scratch

## Final title candidate

Stop prompting coding agents. Build them a loop.

## Description draft

Most people use AI coding agents like chatbots: type a vague prompt, wait for a diff, then babysit the mess.

In this video I break down the simple agentic loop behind useful coding agents, build the loop from scratch, and show how to apply the same pattern in Codex, Claude Code, and OpenCode.

The loop:

1. Goal
2. Context
3. Plan
4. Act
5. Observe
6. Verify
7. Checkpoint

This is not about hype or replacing engineering judgment. It is about making agent work bounded, reviewable, and testable.
