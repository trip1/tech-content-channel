# Platform Setup

## Long-form platforms

| Platform | Purpose | Cadence | Notes |
|---|---|---:|---|
| YouTube | Polished tutorials, recaps, explainers | 1-2/week | Search-friendly titles and thumbnails |
| Twitch | Live coding/building/community | 1-3/week | Archive streams for clips |
| Kick | Live coding/building/community | 1-3/week | Mirror Twitch stream if desired |

## Short-form / distribution

| Platform | Purpose | Cadence | Notes |
|---|---|---:|---|
| YouTube Shorts | Highlights, quick tips, failures/fixes | 3-5/week | Extract from streams/long-form |
| TikTok | Discovery clips | 3-5/week | Strong first 1-2 seconds |
| X/Threads/LinkedIn | Build logs, diagrams, launch posts | as needed | Repurpose notes and screenshots |

## Setup checklist

- [ ] Decide channel name and handles
- [ ] Create brand kit: logo, colors, fonts
- [ ] Create OBS scenes
- [ ] Create thumbnail templates
- [ ] Create intro/outro or recurring bumper
- [ ] Configure stream archive workflow
- [ ] Configure link-in-bio / landing page
