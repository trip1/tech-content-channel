# Project brief: 5 AI Tools That Actually Save Time

## Working title
5 AI Tools That Actually Save Time

## Core promise
A short practical list with one real task per tool and a keep/skip verdict.

## Audience
Busy developers, creators, and tech-curious viewers who want practical AI without hype.

## Viewer transformation
- **Before:** The viewer has heard the topic, but does not know what to do with it or how to judge it.
- **After:** The viewer has a simple mental model, a repeatable checklist, and a low-risk first action.

## Story angle
Most AI tools add another inbox. These five have to pass one test: do they save me time this week?

## Why now
AI tool fatigue is high, so practical filters are valuable.

## Format
- Main platform: YouTube longform
- Length target: 8-11 minutes
- Production style: screen recording, simple desk/phone B-roll, lightweight graphics, minimal talking head
- Skill level: beginner-friendly, practical, honest, no hype unless the demo earns it

## Core beats
1. Show the result/conflict immediately.
2. Explain why the viewer should care now.
3. Do a simple visual or screen-recorded demo.
4. Show the limits, risks, or where the idea breaks.
5. Give a clear recommendation and next step.

## Demo / proof plan
Screen record each tool completing a small real task: summarize notes, draft commands, generate captions, inspect code, automate a file workflow.

## Assets needed
- [ ] Screen recording of setup/demo/result
- [ ] One simple diagram or scorecard
- [ ] 2-3 pieces of desk/keyboard/laptop B-roll
- [ ] Thumbnail screenshot or prop photo
- [ ] Current tool/version screenshots where relevant

## Risks / blockers
- Avoid affiliate-style hype; show actual before/after time saved or friction added.
- Redact personal data, account names, API keys, tokens, passwords, emails, and file paths that should not be public.
- Re-check product names, pricing, and availability on recording day.

## Repurposing plan
- **Short 1:** The one question I use before trying any new AI tool.
- **Short 2:** One AI tool I would keep, and one I would skip.
- **Social post:** One practical checklist or decision rule from the video.
