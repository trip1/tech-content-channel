# Publishing Checklist

## YouTube long-form

- [ ] Title has clear outcome or curiosity gap
- [ ] Thumbnail matches the title promise
- [ ] Description includes links/resources
- [ ] Chapters added
- [ ] End screen/cards added
- [ ] Pinned comment drafted
- [ ] Shorts candidates identified

## Twitch/Kick live

- [ ] Stream title states the goal
- [ ] Category/tags set
- [ ] Going-live post sent
- [ ] Recording/archive enabled
- [ ] Timestamps captured for clips

## Repurposing

- [ ] 3-5 Shorts cut from long video/stream
- [ ] Social post/thread summarizing lesson
- [ ] Blog/repo notes updated if applicable
