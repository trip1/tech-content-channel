# Thumbnail brief: Stop prompting coding agents. Build them a loop.

## Thumbnail job

Make viewers immediately understand that this is not another generic AI tooling video. It is about the *control loop* that makes coding agents useful.

## Best concept

### Build the loop, not the prompt

Visual:

- Dark terminal / mission-control background
- Center: circular loop diagram with four large labels:
  - PLAN
  - ACT
  - VERIFY
  - STOP
- Left side: messy chat bubble: `fix my app`
- Right side: green terminal check: `tests passed`
- Small tool labels along bottom: Codex · Claude Code · OpenCode

Thumbnail text:

```text
BUILD THE LOOP
```

## Alternate concepts

### Concept B: Not a prompt

- Big red X over a prompt box
- Bright flowchart beside it
- Text: `NOT A PROMPT`

### Concept C: Agents that finish

- Agent dashboard with toggles: Context, Tools, Tests, Stop
- Green status: VERIFIED
- Text: `AGENTS THAT FINISH`

## Visual style

- Dark terminal-inspired background
- Cyan/green verification accents
- Yellow warning accent around vague prompt
- Keep text to 2-3 words max
- Use real terminal screenshots as texture, not as readable detail

## Screenshot assets to capture

- Passing test terminal
- Small `AGENTS.md` file open in editor
- Diagram sketch of loop
- Codex / Claude Code / OpenCode terminal windows, cropped so no credentials or private repo info show
