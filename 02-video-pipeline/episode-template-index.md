# Episode Folder Pattern

Create one folder per major video or stream series:

```text
02-video-pipeline/episodes/YYYY-MM-DD-working-slug/
├── brief.md
├── research.md
├── packaging.md
├── script.md
├── shot-list.md
├── edit-notes.md
├── publish-notes.md
├── production-checklist.md
└── assets/
```

Start from `templates/episode-brief.md`, `templates/video-script.md`, and `templates/production-checklist.md`.

For the full idea-to-release workflow, use `02-video-pipeline/video-production-walkthrough.md`.
