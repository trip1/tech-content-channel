# Packaging: Agentic loops from scratch

## Core promise

By the end, viewers can stop treating coding agents like chatbots and start running them like bounded engineering loops.

## Viewer transformation

Before: "I typed a prompt and hoped the agent did the right thing."

After: "I know how to write the loop, the repo instructions, the verification step, and the stop condition."
