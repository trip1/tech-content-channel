# Stream Run of Show

## Pre-stream

- [ ] Confirm stream title/category/tags
- [ ] Check mic/camera/screen capture
- [ ] Open project, notes, docs, terminal
- [ ] Start local recording if needed
- [ ] Post going-live message

## Opening

1. Quick welcome
2. State today's goal
3. Show the finished target or diagram
4. Explain constraints and success criteria

## During stream

- Narrate decisions, not every keystroke
- Capture timestamps for clip moments
- Keep a visible TODO list
- Summarize after major milestones

## Closing

- Show what works now
- List what remains
- Ask for comments/questions
- Mention next stream/video
