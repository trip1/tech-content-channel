# Episode Folder Pattern

Create one folder per major video or stream series:

```text
02-video-pipeline/episodes/YYYY-MM-DD-working-slug/
├── brief.md
├── outline.md
├── script.md
├── shot-list.md
├── edit-notes.md
└── publish-notes.md
```

Start from `templates/episode-brief.md` and `templates/video-script.md`.
