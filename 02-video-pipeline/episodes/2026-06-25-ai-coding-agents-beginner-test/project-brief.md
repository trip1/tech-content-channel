# Project brief: I Tried 3 AI Coding Agents So You Do Not Have To

## Working title
I Tried 3 AI Coding Agents So You Do Not Have To

## Core promise
A practical first impression of Codex, Claude Code, and OpenCode using the same task, judged by setup friction, code quality, verification, and repair behavior.

## Audience
Developers and tech-curious builders who keep hearing about AI coding agents but have not picked one.

## Viewer transformation
- **Before:** The viewer has heard the topic, but does not know what to do with it or how to judge it.
- **After:** The viewer has a simple mental model, a repeatable checklist, and a low-risk first action.

## Story angle
Three agents, one tiny real-world task, and a simple scoreboard: who actually helps a beginner ship?

## Why now
AI coding agents are moving from novelty to daily developer workflow, and terminal-based agent tools are especially visible right now.

## Format
- Main platform: YouTube longform
- Length target: 8-11 minutes
- Production style: screen recording, simple desk/phone B-roll, lightweight graphics, minimal talking head
- Skill level: beginner-friendly, practical, honest, no hype unless the demo earns it

## Core beats
1. Show the result/conflict immediately.
2. Explain why the viewer should care now.
3. Do a simple visual or screen-recorded demo.
4. Show the limits, risks, or where the idea breaks.
5. Give a clear recommendation and next step.

## Demo / proof plan
Use a tiny repo task: add a CLI flag, update README, run tests, then ask each agent to fix one intentionally failing test.

## Assets needed
- [ ] Screen recording of setup/demo/result
- [ ] One simple diagram or scorecard
- [ ] 2-3 pieces of desk/keyboard/laptop B-roll
- [ ] Thumbnail screenshot or prop photo
- [ ] Current tool/version screenshots where relevant

## Risks / blockers
- Tool behavior changes quickly; record exact versions and frame this as a beginner workflow test, not a permanent ranking.
- Redact personal data, account names, API keys, tokens, passwords, emails, and file paths that should not be public.
- Re-check product names, pricing, and availability on recording day.

## Repurposing plan
- **Short 1:** The scoreboard that made one AI coding agent feel safer for beginners.
- **Short 2:** The mistake people make when comparing Codex, Claude Code, and OpenCode.
- **Social post:** One practical checklist or decision rule from the video.
